Allows you to add relevant dynamic content ads from the Prosperent affiliate network.


== Description ==

This plugin uses the Prosperent affiliate networks' API to display relevant ads to your WP site.  If users click through on the ads and buy the product you will earn a commission on the sale.

The plugin offers several built in display templates as well as the ability to add your own custom template.

There are a lot of options that can be set, including the ability to display ads based on the keyword used to find your site in the search engine, thus giving you laser targeted ads.


== Installation ==


1. Upload the `wpprosperent` folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update settings.

== Frequently Asked Questions ==

= Can this be used with adsense? =

Yes, it's fine with adsense or any other ads.
